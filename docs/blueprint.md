# **App Name**: BeeGym Pro

## Core Features:

- Student Management: Manage student data, track progress, and communication logs in the app
- Workout Creation: Design personalized workouts using an extensive library of exercises with multimedia support.
- Automated Exercise Suggestions: Suggest exercises and training programs based on the students' goals and training history using AI tool. Provides instructional media where available.
- Scheduling & Calendar: Automate calendar with scheduler for managing class availability
- Payments Tracking: Monitor payments, send reminders, and manage subscriptions all in an easy-to-use interface. Track revenue from subscriptions and individual sessions. Provide payment options
- Performance Reporting: Analyze student progress and outcomes and present key data through summary reports and visualizations
- Direct Messaging: Send and receive messages and resources from clients via direct messages

## Style Guidelines:

- Primary color: Vibrant cyan (#2DD4FF), reflecting the energy and innovation of modern fitness training.
- Background color: Light cyan (#CFFAFE), providing a clean, fresh, and calming backdrop.
- Accent color: Electric purple (#9F7AEA), used sparingly to highlight interactive elements and important information.
- Headline font: 'Space Grotesk' sans-serif, lending a futuristic and digital feel.
- Body font: 'Inter' sans-serif, used to ensure readability and clarity.
- Material Symbols Outlined for a consistent and modern look.
- Clean and intuitive layout with a focus on easy navigation. Use clear sections for managing students, workouts, and payments.
- Subtle transitions and animations to enhance user experience.
